==========
OSInt
==========

Install

This package is not available in pypi right now, so you need to install
it using github.

Clone repository

    git clone https://github.com/bibi21000/sphinxcontrib-osint.git
Change to directory

    cd sphinxcontrib-osint
Make venv and install

    python3 -m venv venv
    ./venv/bin/pip install .
Example
Build example demo

    cd examples
    make html
